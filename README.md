# Intelligent Disease Prediction and Doctor Recommendation System

This is a complete, production-ready full-stack B.Tech Final Year project. It uses Machine Learning to predict potential diseases based on user-provided symptoms and recommends relevant medical specialists with direct links for consultation.

## Features

- **Machine Learning**: Predicts diseases using an ensemble of 3 models (Decision Tree, Random Forest, Naive Bayes).
- **Comprehensive Disease Profiles**: Enriches the prediction with detailed descriptions, recommended diets, required medications, safety precautions, and suggested workouts dynamically loaded from diverse medical datasets.
- **Voice Input**: Users can speak their symptoms using the Web Speech API.
- **Doctor Recommendation**: Automatically maps the predicted disease to a medical specialist and provides links to Apollo 24/7 and Practo.
- **Interactive UI**: Built with React, Tailwind CSS (utilizing the sleek Inter font), and Recharts for a modern, responsive, and beautiful experience.
- **AI Health Assistant**: A built-in chatbot for basic health queries.
- **History Tracking**: Saves previous predictions locally for easy access.

## Project Structure

- `backend/`: FastAPI Python server and Machine Learning models.
- `frontend/`: React + Vite frontend application.

## Prerequisites

- Node.js (v16 or higher)
- Python (v3.9 or higher)

## Setup Instructions

### 1. Backend Setup

1. Open a terminal and navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Install the required Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. (Optional) Re-train the models and generate the dataset. *Note: Pre-trained models are already included, so this step is only necessary if you modify the diseases or symptoms.*
   ```bash
   python train.py
   ```
4. Start the FastAPI server:
   ```bash
   python app.py
   ```
   *The backend server will run at `http://localhost:8000`.*

### 2. Frontend Setup

1. Open a new terminal and navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
2. Install the required NPM dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
   *The frontend application will be accessible at `http://localhost:5173`.*

## How to Use

1. Navigate to the frontend URL in your browser.
2. Click **Start Diagnosis** or click **Login** in the top right to create a local account.
3. On the Predict page, select your symptoms from the searchable list or use the **Microphone** icon to speak them.
4. Click **Predict Disease** to see the top 3 predictions, model comparison charts, doctor recommendations, and the newly added **Comprehensive Disease Profile** (detailing diets, medications, and workouts).
